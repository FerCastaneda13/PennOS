Implementation of linked list functions. 

This file contains the implementation of functions declared in \mbox{\hyperlink{linked__list_8h_source}{linked\+\_\+list.\+h}}. It includes functions for creating and manipulating a doubly linked list, such as adding, removing, and searching for nodes, as well as managing the list itself.
