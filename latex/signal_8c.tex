\doxysubsection{Detailed Description}
Implementation of signal handling functions for Penn\+OS. 

Created by Omar Ameen on 11/10/23. This file contains the implementations of the functions defined in \mbox{\hyperlink{signal_8h_source}{signal.\+h}}. It includes functions for adding signals, waiters, and timers, as well as retrieving waiting processes based on PID or \mbox{\hyperlink{structPCB}{PCB}} pointers.
